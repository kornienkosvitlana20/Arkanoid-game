# Діаграма варіантів використання (Use Case Diagram)

## Актори
- **Гравець** — людина, що грає
- **Система** — ігровий рушій

## Варіанти використання

```
┌─────────────────────────────────────────────────────────┐
│                     Arkanoid System                     │
│                                                         │
│  ┌─────────────────┐    ┌──────────────────────────┐   │
│  │  Запустити гру  │    │ Налаштувати параметри CLI │   │
│  └─────────────────┘    └──────────────────────────┘   │
│         ▲                          ▲                    │
│  ┌─────────────────┐    ┌──────────────────────────┐   │
│  │  Керувати       │    │ Поставити на паузу        │   │
│  │  платформою     │    └──────────────────────────┘   │
│  └─────────────────┘                                   │
│         ▲                                               │
│  ┌─────────────────┐    ┌──────────────────────────┐   │
│  │  Запустити м'яч │    │ Перезапустити гру         │   │
│  └─────────────────┘    └──────────────────────────┘   │
│                                                         │
│  ┌─────────────────┐    ┌──────────────────────────┐   │
│  │ Збирати очки    │    │ Перейти на наступний     │   │
│  │ (авто)          │    │ рівень (авто)            │   │
│  └─────────────────┘    └──────────────────────────┘   │
└─────────────────────────────────────────────────────────┘
            ▲
       [Гравець]
```

## Опис варіантів

| UC | Назва | Актор | Опис |
|----|-------|-------|------|
| UC1 | Запустити гру | Гравець | Запуск з можливими аргументами CLI |
| UC2 | Налаштувати параметри | Гравець | `--difficulty`, `--bg-color`, `--lives`, `--fullscreen` |
| UC3 | Керувати платформою | Гравець | Клавіші ← →, мишка |
| UC4 | Запустити м'яч | Гравець | Пробіл або клік миші |
| UC5 | Поставити на паузу | Гравець | Клавіша P |
| UC6 | Перезапустити гру | Гравець | Клавіша R після game over |
| UC7 | Збирати очки | Система | Нарахування при знищенні цеглини |
| UC8 | Перейти на рівень | Система | Автоматично після зникнення всіх цеглин |
